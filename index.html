About Me

Hi, I'm Eric Chen. I'm currently a senior at Amador Valley High School. This website is meant to serve as a portfolio for the art I have created throughout the first semester of my Digital Art class. So far, we've learned how to utilize Adobe Indesign and Illustrator as well as how to incorporate design elements such as typography, color, and spacing into our art.

My Indesign Projects

Lizard Drawing

This lizard drawing helped me learn many key fundamentals of Indeisgn. For example, it taught me how to utilize layers and properly trace objects with the linetool. Furthermore, I learned about special effects such as applying drop shadow and a glow to objects. I also learned how to create a gradient color when making the background.

Turtle Drawing

This turtle drawing served primarily as additional linetool practice. The different, sharper shapes that are present within the drawing made it more difficult. For example, the tongue of the turtle was especially difficult to accurately create with the linetool. This project also served as further practice for properly utilizing layers. For example, the shell of the turtle is on a layer above the body of the turtle because the shell of the turtle is meant to appear on top.

Birthday Card

This birthday card had the restriction where we were only allowed to use typography to create the design. This restriction forced me to get creative with the design. For example, the drops of wine seen in the image aren't circles, but are rather the letter O with an increased stroke length that caused the letter to become filled and appear as a circle. This project also taught me the basics of typography, such as kerning, tracking, and leading.

My Illustrator Projects

Pumpkin Drawing

This pumpkin drawing allowed me to practice the skills I learned for Adobe Illustrator. For example, I utilized the revolve tool in order to create the 3D shape present within the stem of the pumpkin. I also used the extrude tool on the body of the pumpkin. In order to create the outline of the pumpkin itself, I also used a join path tool to form the holes within the pumpkin.

Vector Face Drawing

This drawing was created by taking an image of myself and recreating it to the best of my ability. I learned new useful tools, such as the mirroring tool which allows me to only create one side of a symmetrical shape. In this case, I mirrored my face across the center. Additionally, I also learned how to incorporate shading in my drawings. I traced a portion of the left side of my face with the linetool and then made it a shade of grey with a low opcaity in order to shade it.

My Final Project

Christmas Tree

This final project was an accumulation of all the knowledge I've gained throughout the semester. For example, the trunk was made using the revolve tool, the seperate layers of the leaves were made using a transform tool, and the shadows underneath the tree were created with a low opacity radial gradient. The star was also created using the glow effect.

House

The house also utilized a considerable amount of techniques from previous projects. For example, the mistletoe uses both the join path tool and the masking tool in order to have the leafy texture on the overall shape. Portions of the icicles were also created using the transform tool, and the doorknob was created with a revolve effect on a circle which was then rotated to face the correct direction.

Final Design

This image is the fully completed final design of this semester. I utilized a custom brush tool with randomized spacing and size in order to create the effect of snow within the snowglobe. I utilized the revolve tool in order to create the base of the snowglobe, and create the window and glass of the snowglobe itself with a low opacity sky blue color. I also used the brush tool for the snow on the ground outside. By using the brush tool, I created a smoother and more even snowy texture on the ground.
